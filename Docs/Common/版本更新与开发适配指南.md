# SDK 3.30.305

## iOS

### 1. 移除(instancetype)instance

TMAccountService 、TMInMeetingService、TMPreMeetingService、TMUserConfigService 接口中全部去掉了`+ (instancetype)instance; `接口, 后续需要实例这个类需要使用 TencentMeetingSDK 类的实例方法获取，分别为：`getAccountService`，`getInMeetingService`，`getPreMeetingService`，`getUserConfigService`。 

比如原来使用代码为：`[TMAccountService instance]`; 

现在改为 `[[TencentMeetingSDK instance] getAccountService]`，
其他的同理。
